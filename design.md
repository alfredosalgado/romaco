# Guía de Diseño - Romaco SPA

## Paleta de Colores

### Color Principal
- **Amarillo/Naranja:** `#fab012` 
  - Usado en elementos destacados, separadores, botones CTA
  - Color corporativo principal
  - Aplicado en fondos de secciones importantes

### Colores Secundarios
- **Blanco:** `#ffffff`
  - Fondo principal de secciones
  - Tarjetas de productos
  - Elementos de contenido

- **Gris Claro:** `#ecebeb`
  - Fondo de secciones alternativas
  - Separación visual entre contenidos

### Esquema de Colores por Sección

#### Header/Navegación
- Fondo blanco con elementos en amarillo corporativo

#### Secciones de Productos
- Fondo blanco para tarjetas de productos
- Títulos y separadores en amarillo `#fab012`
- Texto en colores neutros

#### Call-to-Action
- Fondo amarillo `#fab012` para destacar
- Texto en blanco para contraste

#### Footer
- Colores neutros con acentos en amarillo

## Elementos Visuales

### Separadores
- Tipo: Líneas pequeñas centradas
- Color: `#fab012`
- Grosor: 4px
- Ancho: 48px
- Espaciado superior: 19px
- Espaciado inferior: 39px

### Tipografía
- Uso de jerarquías claras con H1, H2, H3
- Títulos principales destacados
- Subtítulos con separadores visuales

### Layout
- Diseño en grid de 3 columnas para productos
- Secciones full-width alternadas
- Espaciado consistente entre elementos

### Imágenes
- Iconos representativos para cada categoría de producto
- Imágenes centradas en tarjetas
- Slider principal en homepage

## Estilo Visual
- **Estilo:** Limpio y profesional
- **Enfoque:** Industrial/construcción
- **Contraste:** Alto contraste con amarillo sobre blanco
- **Espaciado:** Generoso uso del espacio en blanco
- **Estructura:** Grid organizado y simétrico
